The homepage of Guruji Astro

## Tech Stack
**Frontend** : React , SCSS, react-slick ,slick-carousel, Swiper

## Run Locally

Clone the project

```bash
  git clone https://github.com/shoaib9430/GURUJI_ASTRO-Frontend_Developer
```

Install dependencies

```bash
  npm i
```

Start the server

```bash
  npm run start
```
